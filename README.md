# Clock
A clock app.<br>
UI/UX inspired by Samsung Clock.
## Build
- Initialize Python environment (and activate it).
- Install packages: `pip install -r requirements.txt`
- To run: `python src/main.py`
- To build (on Linux): `source build.sh`
  - Binaries are generated in `build/main.dist`.
